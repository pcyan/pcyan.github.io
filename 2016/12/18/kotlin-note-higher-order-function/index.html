前言

本篇讲的是kotlin中高阶函数的使用及技巧。

什么是高阶函数

在kotlin中，变量不仅仅可以是基本类型和引用类型，还可以是一个函数，函数可以被当作参数传递，
或者作为返回值被返回，这样的函数称为高阶函数。
